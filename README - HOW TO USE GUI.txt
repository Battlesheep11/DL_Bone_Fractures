Instructions:
First, download and extract the zip together into a folder (this is important!)
Install the necessary libraries. the script requires a few Python packages to run. Open a terminal and run the following command:
pip install torch torchvision pillow numpy
Navigate to the project folder in the terminal.
A simple way to do this is to type cd  (with a space after cd), then drag and drop the folder from your File Explorer into the terminal window, and press Enter.
Run the script by executing the following command in the terminal:
python gui_infer.py
